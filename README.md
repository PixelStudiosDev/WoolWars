## Developer API

### Maven
```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```
```xml
<dependency>
    <groupId>com.github.CubeCrafter</groupId>
    <artifactId>WoolWars</artifactId>
    <version>Tag</version>
    <scope>provided</scope>
</dependency>
```
### Gradle
```groovy
repositories {
    maven { url 'https://jitpack.io' }
}
```
```groovy
dependencies {
    compileOnly 'com.github.CubeCrafter:WoolWars:Tag'
}
```
